# Changelog - Cat Clicker

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-21

### 🎉 Initial Release

#### Added
- ✨ 7 levels with progressive difficulty
- 🎨 2 visual themes (Pink & Blue)
- 🐱 Cute cat character with animations
- 🖱️ Click detection and scoring system
- ⭐ Multiplier system (x2, x3 bonuses)
- 🔊 Sound effects for interactions
- 💾 Score persistence system
- 🎯 Special challenges and secret bonuses
- ⚡ 60 FPS performance optimization
- 📊 Leaderboard integration

#### Features
- Game state management
- Level progression tracking
- Theme switching
- Score calculation and storage
- Sound effects library
- UI and menu system

#### Technical
- Built with C (ISO C11)
- SDL2 for graphics and audio
- Cross-platform support (Windows, macOS, Linux)
- CMake build system
- Comprehensive documentation

## Version Format

We follow [Semantic Versioning](https://semver.org/):
- **MAJOR**: Incompatible API changes
- **MINOR**: Backward-compatible functionality additions
- **PATCH**: Backward-compatible bug fixes

---

## Release Strategy

- **Stable Releases**: Monthly
- **Hotfixes**: As needed
- **Beta Releases**: Pre-release versions for testing

---

## How to Report Changes

When contributing, update this file with:
1. Version number
2. Date (YYYY-MM-DD format)
3. Changes categorized as:
   - Added
   - Changed
   - Deprecated
   - Removed
   - Fixed
   - Security

Example:
```markdown
## [1.1.0] - 2026-05-21

### Added
- New feature description

### Fixed
- Bug fix description

### Changed
- Behavioral change description
```
