# Quick Start - Cat Clicker 🐱

Get started with Cat Clicker in 5 minutes!

## ⚡ 30-Second Setup

```bash
# 1. Clone
git clone https://github.com/maryambenzekri/cat-clicker.git
cd cat-clicker

# 2. Build (choose your OS)
cmake -B build && cd build && cmake --build .
# OR on macOS with Homebrew
brew install sdl2 && cmake -B build && cd build && cmake --build .
# OR on Windows (if SDL2 installed)
gcc cat_clicker.c -lSDL2 -lm -o cat_clicker

# 3. Play
./cat_clicker
```

Done! 🎮

## 🛠️ Installation Issues?

### Problem: "SDL2 not found"

**Quick Fix (macOS):**
```bash
brew install sdl2
```

**Quick Fix (Ubuntu/Debian):**
```bash
sudo apt install libsdl2-dev
```

**Quick Fix (Windows):**
- Download SDL2 from https://www.libsdl.org/download-2.0.php
- Add to your PATH or compiler flags

### Problem: "Compiler error"

**Solution:** Use a different compiler
```bash
# Try Clang instead of GCC
clang -o cat_clicker cat_clicker.c -lSDL2 -lm
```
