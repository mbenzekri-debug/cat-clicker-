## 🚀 Étapes pour Lancer le Repository

### 1️⃣ Créer le Repository sur GitHub
```bash
# Allez sur https://github.com/new
# Repository name: cat-clicker
# Description: 🐱 A cute clicker game in C with SDL2
# Public
# Ne pas initialiser (on va pusher local)
```

### 2️⃣ Pousser le Code Local
```bash
cd "c:\Users\amity\OneDrive\Desktop\cat game"

# Initialiser Git
git init
git add .
git commit -m "Initial commit: Cat Clicker game project"

# Ajouter le remote (remplacer USERNAME)
git remote add origin https://github.com/USERNAME/cat-clicker.git
git branch -M main
git push -u origin main
```

### 3️⃣ Configurer le Repository (GitHub)
- Allez à Settings
- Activez Discussions
- Activez Wiki
- Ajoutez topics: gamedev, c, sdl2, clicker-game, education
- Configurez branch protection pour main

### 4️⃣ Inviter les Collaborateurs
- Allez à Settings → Collaborators
- Invitez Maryam Benzekri et Chada Hennouk
